## 📝 http_static-server

### 💡 How to create a Server project like this?
- Run `yarn global add ts-node-dev` to install tsnd
- Run `npm/yarn init -y` to create `package.json` in your project

### 🔨 Some relative tools
- node-dev: Automatically restarts the node process when a file is modified
- ts-node: A TypeScript execution engine and REPL for Node.js
- **ts-node-dev**: A tool for restarting ts-node services based on node-dev.
